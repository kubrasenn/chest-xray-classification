# Chest X-ray Classification (Python / PyTorch)

A PyTorch-based project for classifying chest X-ray images (e.g., Normal vs. Pneumonia/COVID depending on the dataset).  
This repo demonstrates an end-to-end workflow: data loading → preprocessing → training → evaluation.

## Why this project
I built this project to strengthen my practical skills in:
- Python data pipelines for image datasets
- training and evaluating deep learning models in PyTorch
- reproducible experimentation and clear documentation

## Tech Stack
- Python
- PyTorch
- NumPy, Pandas
- Matplotlib / Seaborn
- Jupyter (optional)

## Dataset
This project uses a chest X-ray dataset (Kaggle / public dataset).  
**Note:** Please follow the dataset license/terms. I do not redistribute the dataset in this repository.
